# casebook

個人的な意思決定・検討プロセスを「事例集」として蓄積するためのリポジトリ。

- カメラ／ガジェットの購入判断
- 子育てフェーズにおける機材・デバイス選定
- 実家見守りカメラ
- 携帯電話運用設計
- リニアトラック設計
- 運用・失敗・妥協点の記録

**結論だけでなく、判断に至る過程を残す**ことを目的とする。

---

## 方針

- 1事例 = 1ファイル（Markdown）
- 時系列で残す（後から見返せることを重視）
- 正解を押し付けない
- 当時の前提条件・制約を明示する

## 事例フォーマット

基本的に以下のテンプレートを使用する。

```
結論:
使用目的:
優先順位（高→低）:
比較軸:
判断理由:
妥協点:
購入/選定の条件:
適用範囲:
```


テンプレートは `templates/case-template.md` を参照。

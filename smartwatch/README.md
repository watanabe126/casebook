## このカテゴリで重視する観点

- **常時装着できるか**
  - 重量
  - 睡眠時の違和感
  - 手首以外（足首・指輪等）の選択肢

- **バッテリー運用**
  - 毎日充電が必要か
  - 実使用で何日もつか
  - 劣化後の使い勝手

- **ライフログの質**
  - 睡眠・心拍・HRV・活動量
  - 「正確さ」よりも「傾向把握」に向くか

- **母艦との関係**
  - iOS / Android での制約差
  - 通知の安定性
  - アプリの成熟度

- **セキュリティ・データ管理**
  - クラウド依存度
  - データの可搬性・削除可否
  - メーカーの透明性
